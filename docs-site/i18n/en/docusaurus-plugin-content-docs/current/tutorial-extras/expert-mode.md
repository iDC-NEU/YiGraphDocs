# Case 3. Expert Mode

## Input Instructions
This mode requires you to provide specific execution steps and graph algorithms. It is recommended to first understand the various graph algorithms supported by EtuGraph. EtuGraph will strictly execute according to your instructions and generate an analysis report.

Example Question: It is initially determined that the transaction network has a high risk level, but currently only 1 account can be frozen as a priority. The goal is to break up the high-risk network to the greatest extent and cut off suspicious fund flows as much as possible.

Instructions Input to EtuGraph:
```
First, calculate the node with the highest global influence in the graph. Second, evaluate the network fragmentation effect (the number of connected components after removing the node).
```
Wait for EtuGraph to run and generate an analysis report.
